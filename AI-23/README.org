** Improper Syntax
#+begin_quote
Any =loop= and =condition= MUST have a =:=. \\
Calling a =function= MUST contain =()= /(may have parameters)/
#+end_quote
#+begin_src python
for i in range(5):      # note the :
    if True:            # note the :
        ultra = precise_results(0)      # note the ()

** = OR ==
#+begin_quote
~=~ assigns a variable BUT \\

#+end_quote

    print('hello, world!')
